# OIBSIP_Data_Analytics_T1_EDA-Retail-Sales
#  Retail Sales Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer purchasing behavior, and product performance. The insights generated can help businesses make data-driven decisions.

## Objectives
- Clean and prepare the dataset
- Perform descriptive statistical analysis
- Analyze sales trends over time
- Explore customer and product behavior
- Create meaningful visualizations
- Provide business recommendations

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Visualizations
- Gender Distribution
- Age Distribution
- Product Category Sales
- Monthly Sales Trend
- Revenue by Gender
- Correlation Heatmap
- Total Amount Distribution
- Top 10 Highest Value Transactions
- Product Category Revenue Pie Chart
- Monthly Transaction Count

## Key Insights
- Clothing generated the highest revenue.
- Female customers contributed slightly more sales.
- Monthly sales showed seasonal fluctuations.
- A small number of transactions generated a significant share of revenue.

## Recommendations
- Focus marketing on top-selling product categories.
- Retain high-value customers through loyalty programs.
- Increase promotions during low-sales periods.
- Use sales trends for better inventory planning.
